# ShoesPriceGraph

## 项目简介
web开发技术的学习与实践，记录球鞋价格变化的网站，能够通过名称模糊搜索，查看球鞋的图片以及价格变化的折线图，前端基于Bootstrap，后端基于Flask，折线图则使用pyecharts，每天自动从网上爬取球鞋的包括名称、颜色、型号、大小等数据，使用MySQL存储数据。
